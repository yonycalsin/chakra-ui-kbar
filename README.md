# Chakra Ui Kbar

## Getting Started

```bash
# with npm
npm install -S -E chakra-ui-kbar@latest

# with yarn
yarn add -S -E chakra-ui-kbar@latest

# with pnpm
pnpm add -S -E chakra-ui-kbar@latest
```

```tsx
import { ChakraKBarProvider } from 'chakra-ui-kbar'

function MyApp() {
  return <ChakraKBarProvider>// ...</ChakraKBarProvider>
}
```
